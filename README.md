# HardwareExplorer

Flutter Project PAS XI DKV B SMK Negeri 4 Malang

## Project Overview

HardwareExplorer is a Flutter application designed as a learning medium for basic computer hardware concepts. This project was created for PJBL (Project-Based Learning) purposes at SMK (Vocational High School). It aims to provide students with an interactive and easy-to-understand platform to learn about computer components.

## Key Features & Benefits

-   **Educational Content:** Comprehensive learning materials on computer hardware components such as CPU, RAM, Storage, and more.
-   **Interactive Quizzes:** Module-based quizzes to reinforce learning.
-   **User-Friendly Interface:** Simple and intuitive UI designed for easy understanding.
-   **Flutter Based:** Cross-platform compatibility with Android, iOS, and Web support.

## Prerequisites & Dependencies

Before you begin, ensure you have the following installed:

-   **Flutter SDK:** [Install Flutter](https://flutter.dev/docs/get-started/install)
-   **Dart SDK:** Comes bundled with Flutter SDK.
-   **Android Studio/Xcode:** For building native Android/iOS apps (optional if only using web).
-   **Web Browser:** For web development and testing.

## Installation & Setup Instructions

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/AxelionAxell/HardwareExplorer.git
    cd HardwareExplorer
    ```

2.  **Install dependencies:**

    ```bash
    flutter pub get
    ```

3.  **Run the application:**

    *   **Android:**

        ```bash
        flutter run -d android
        ```

    *   **iOS:**

        ```bash
        flutter run -d ios
        ```

        *Note: Requires Xcode and iOS simulator or connected device.*

    *   **Web:**

        ```bash
        flutter run -d chrome
        ```

## Usage Examples & API Documentation

This project primarily focuses on UI/UX and educational content. There are no external APIs used. The application is designed to be intuitive; navigate through the different modules to access the learning material and quizzes.
